    1. While writing code:
        1. Avoid abbreviating variables; define them close to their usage.
        2. Debug effectively.
        3. Functions are for one thing.
        4. Indent code properly.
        5. Keep it simple.
        6. Limit function arguments (Least is 3).
        7. See the syntax.
        8. Simplify the conditional expressions.
        9. Update the code frequently.

    2. To learn a new language:
        1. Research/Find the differences.
        2. Find features you don't understand.
        3. Follow a short tutorial.
        4. Practice with programming problems.
        5. Look up troubling topics.
        6. Repeat step 4 & 5 until comfortable.
        7. Work on a small project.
